# Inflation Dynamics in Europe (2020–2024)

This repository contains an independent, self-initiated data analysis project examining inflation dynamics in selected European economies between 2020 and 2024.

## Objective
To analyze inflation trends, cross-country differences, distributional properties, and volatility using descriptive statistics and visualization techniques.

## Data
Annual inflation rate data for Germany, France, Italy, Switzerland, Czechia, Spain, and the Netherlands.

## Methods
- Descriptive statistics (mean, standard deviation)
- Time-series and cross-sectional visual analysis
- Python (Pandas, Seaborn, Matplotlib)

## Structure
- data/: Raw data
- notebooks/: Jupyter notebook with analysis
- figures/: Final figures used in the report
- report/: 2-page PDF report

## Note
This project was conducted independently and is not part of any university coursework.
